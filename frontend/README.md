# Stock Trading Using Vue


## Feature

``` bash
  1. Display stock in graph
  2. Buy Stock
  3. Sell Stock
  4. Portfolio page
```

## Prerequisite

``` bash
  # Recommend
  node: 8.10.0
  yarn: 1.5.1
```

## Build Setup

``` bash
# install dependencies
yarn

# serve with hot reload at localhost:8080
yarn dev

# build for production with minification :Doesn't set prod, dev, staging
yarn build
```
